Jenkins Project

Jenkins is an open-source automation server widely used for continuous integration and continuous delivery (CI/CD). When combined with cloud platforms like AWS, Azure, or Google Cloud, Jenkins enables teams to automate the entire software development lifecycle from building and testing to deploying applications at scale
